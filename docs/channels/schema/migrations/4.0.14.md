# Migration Notes: Version 4.0.14

**Release Date:** 2026-01-14  
**Version Type:** Minor version bump (UI branch promotion)

---

## Overview

Version 4.0.14 represents the promotion of the 4.0.13-ui UI branch to a stable release version. All UI components, model specifications, and related doctrine that were previously labeled as 4.0.13-ui now belong to 4.0.14.

---

## Version Promotion

### Branch Status

- **4.0.13-sys**: Complete and stable (system architecture branch)
- **4.0.13-ui**: Promoted to **4.0.14** (UI-focused branch)
- **4.0.14**: Current stable version for all UI work

### Version Updates

All version references have been updated to 4.0.14:

- `config/global_atoms.yaml` - Updated `GLOBAL_CURRENT_LUPOPEDIA_VERSION` to `4.0.14`
- `lupo-includes/version.php` - Updated all version constants to `4.0.14`
- `CHANGELOG.md` - Added new section for 4.0.14
- All documentation files - Updated headers to reflect 4.0.14

---

## Database Changes

**No database schema changes** in this version. This is a version promotion only.

---

## UI Components

All UI components, model specifications, and related doctrine that were part of the 4.0.13-ui branch are now part of 4.0.14.

---

## Migration Steps

### For Existing Installations

1. **No database migration required** - This is a version bump only
2. **Update version constants** - Ensure all version references point to 4.0.14
3. **Update documentation** - Review and update any custom documentation to reflect 4.0.14

### For New Installations

Install using the 4.0.14 release package. All UI components and model specifications are included.

---

## Backward Compatibility

- **Fully backward compatible** with 4.0.13-sys
- **No breaking changes** from 4.0.13-ui promotion
- **Legacy files preserved** - All compatibility shims maintained
