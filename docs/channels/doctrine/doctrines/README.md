## üö´ 5. Prohibited Operations

### No Doctrine Modification

Implementations must **not**:
- Modify core doctrine principles
- Create alternative protocol versions
- Use non-canonical axis definitions

- Use deprecated protocol versions
- Override registry definitions
- Implement partial doctrine compliance
