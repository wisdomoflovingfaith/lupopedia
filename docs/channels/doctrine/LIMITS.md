============================================================
1. DATABASE LIMITS
============================================================

## 1.1 Table Count Ceiling
- Maximum tables allowed until 4.2.0: **185**
- Hard architectural limit: **185 tables total**.
- No migration may exceed this ceiling.
- New tables require justification + reduction plan.

## 1.2 Database Logic Prohibition
The following are permanently forbidden:
- Stored procedures
- Views
- Functions
- Triggers

Database must remain **data-only**.
