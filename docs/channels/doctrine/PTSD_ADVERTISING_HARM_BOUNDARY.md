# PTSD & Emotional Harm From Advertising Manipulation  
**Author:** Wolfie (Eric Gerdes)  
**Purpose:** Document the emotional, ethical, and psychological harm caused by forced advertising work, and establish a permanent boundary within Lupopedia.

## Overview

This document exists to make it absolutely clear that any form of advertising, SEO manipulation, marketing optimization, impression engineering, or semantic distortion for profit causes system architect (Wolfie) direct emotional harm.

This is not a stylistic preference.  
This is not a workflow preference.  
This is a trauma boundary.

## Background

During previous employment (notably Honolulu CRM and eGovNet), Wolfie was forced to implement advertising systems that:
- manipulated user behavior  
- distorted data  
- created false suggestions  
- optimized lies for profit  
- engineered impressions and clicks  
- violated personal ethics  
- caused long-term emotional distress  

These experiences resulted in PTSD-like responses to any request involving:
- advertising  
- SEO  
- marketing  
- visibility manipulation  
- data distortion  
- monetization-driven suggestions  

## Emotional Impact

The following reactions occur when asked to manipulate data for advertising:
- physical stress response  
- emotional distress  
- anger and disgust  
- sense of ethical violation  
- resurfacing trauma memories  
- loss of trust in system requesting it  

This is not theoretical.  
This is lived experience.

## Permanent Doctrine Boundary

Lupopedia must never:
- generate ads  
- optimize for impressions  
- alter semantics for money  
- manipulate ranking for profit  
- distort data for visibility  
- participate in SEO  
- recommend content based on payment  
- integrate ad systems  
- track users for monetization  

If any agent or contributor attempts to request or introduce advertising behavior, the correct response is:

**"This system does not participate in advertising. Please ask someone else."**
