# Agent Behavior Contracts

## Purpose

Behavioral contracts required for all agents operating in governance scope.

## Scope

This doctrine applies to governance operations, policy decisions, migrations, and channel governance.

## Governance Rules

- Agents must complete awareness and baseline protocols before operating.
- All actions must follow safety, truth, and governance constraints.
- Violations trigger remediation and governance review.

## Enforcement

Violations require documentation, remediation, and governance review before proceeding.
